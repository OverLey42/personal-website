## Installation

#### requirements

- node 16+ (<kbd>v19.2.0</kbd>)
- npm (<kbd>v8.19.2</kbd>)


### Install the repository on your local device

```console
$ git clone https://github.com/OverLey42/personal-website
$ cd personal-website
```

### Install the projekt (packages)

```console
$ npm i
```

### Start the website

```console
$ npm run dev
```
Website: [localhost](http://localhost:5173/))
